# CV_online_project

This repository hosts the source code for a personal portfolio website based on the **iPortfolio** template by BootstrapMade.

## Template Details

- **Template Name:** iPortfolio  
- **Template URL:** [https://bootstrapmade.com/iportfolio-bootstrap-portfolio-websites-template/](https://bootstrapmade.com/iportfolio-bootstrap-portfolio-websites-template/)  
- **Author:** BootstrapMade.com  
- **License:** [https://bootstrapmade.com/license/](https://bootstrapmade.com/license/)

## Project Structure

- `app.py`: Main Python application (likely Flask or similar web framework)  
- `static/`: Static files such as CSS, JS, images, favicon.ico  
- `templates/`: HTML template files  
- `PWEB_env/`: Virtual environment or dependencies folder  
- `Dockerfile`: Containerization file for deployment  
- `Procfile`: Process file for deployment (Heroku or similar)  
- `.gitignore`: Git ignore rules  
- `requirements.txt`: Python dependencies list  
- `favicon.ico`: Website favicon  
- `Readme.txt`: Original readme file from the template

## Description

This project is a personal online CV/portfolio built using the iPortfolio Bootstrap template. It integrates Python backend (Flask or similar) to serve the website dynamically and is ready for deployment with Docker and cloud platforms.

## Installation & Deployment

1. Clone the repo  
2. Create and activate your Python environment  
3. Install dependencies with `pip install -r requirements.txt`  
4. Run the app with `python app.py`  
5. Access the website at `http://localhost:5000` (default Flask port)  
6. For Docker deployment, build the image with `docker build -t cv_online_project .` and run accordingly
